# Vakit (working name) — Project Docs

Prayer companion app for Muslims in Western countries. Wins on **reliability, accuracy, and trust** — the three documented failures of Muslim Pro (ads + data scandal + apathy) and Pillars (notification/location/calculation bugs).

## Documents

1. **01-product-overview.md** — Vision, positioning, target market, success metrics
2. **02-pain-points-research.md** — Evidence base: validated pain points from Trustpilot/app reviews, severity-ranked
3. **03-prd-mvp.md** — Feature specs, MVP scope, what's deliberately excluded
4. **04-technical-architecture.md** — Stack, on-device privacy architecture, notification engine design, risks
5. **05-monetization.md** — Free-forever worship rule, Vakit Plus tier ($3.99/mo), sadaqah mechanics, projections
6. **06-roadmap-90-days.md** — Phased build plan with exit criteria and kill checkpoints
7. **07-launch-marketing.md** — Ramadan 2027 timing, channels, ASO, mosque partnerships

## The one-paragraph thesis

Muslim Pro has ~180M downloads, permanent trust damage, and ads inside worship. Pillars proved "privacy-first" demand but ships unreliable notifications. Hallow proved faith apps can do $40M/yr. Nobody has built the Muslim prayer app that treats the athan notification as mission-critical infrastructure. That app, launched 6 weeks before Ramadan 2027 with a fair $3.99 premium tier and worship free forever, is the plan.
